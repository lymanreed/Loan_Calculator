Types

Let's say there's a script process_four_numbers.py. It takes exactly four arguments, all of which are numerical. It may be run from the command line like this: python3 process_four_numbers.py 25 17 808 9

Inside the program itself, the arguments are firstly read from the command line into the list sys.argv. You don't see that part of the script but you have access to sys.argv (see below).

Your task is to access the arguments and print them as a list. Note that all arguments must be integers!

Remeber, what does sys.argv store as the first element? And what type do all its elements have?
